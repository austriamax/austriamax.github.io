Tarea 5. Pregunta 10.2.

Calcular la medida asociada a cada \(g_i\) de los siguientes intervalos.

  1. \((0,2]\)

  2. \([0, \frac{1}{2}]\)

  3. \((-\infty, 0]\)

  4. \((3,10]\).

1. \(g_1(x) = 2x\).

  1. \[\lambda_{g_1}((0,2]) = 2(2) - 2(0) = 4.\]

  2. \[\lambda_{g_1}([0, \frac{1}{2}]) = 2(\frac{1}{2}) - 2(0) = 1.\]

  3. \[\lambda_{g_1}((-\infty, 0]) = 2(0) - \lim_{x \to -\infty} 2x = \infty.\]

  4. \[\lambda_{g_1}((3,10]) = 2(10) - 2(3) = 4.\]

2. \(g_2(x) = \arctan{(x)}\).

  1. \[\lambda_{g_2}((0,2]) = \arctan{(2)}- \arctan{(0)} = \arctan{(2)}.\]

  2. \[\lambda_{g_2}([0, \frac{1}{2}]) = \arctan{(\frac{1}{2})} - \arctan{(0)} = \arctan{(\frac{1}{2})}.\]

  3. \[\lambda_{g_2}((-\infty, 0]) = \arctan{(0)} - \lim_{x \to -\infty} \arctan{(x)} = 0 - (-\frac{\pi}{2}) = \frac{\pi}{2}.\]

  4. \[\lambda_{g_2}((3,10]) = \arctan{(10)} - \arctan{(3)}.\]

3. \(g_3(x) = 1_{[0, \infty)}(x)\).

  1. \[\lambda_{g_3}((0,2]) = 1_{[0, \infty)}(2) - 1_{[0, \infty)}(0) = 1 - 1 = 0.\]

  2. \[\lambda_{g_3}([0, \frac{1}{2}]) = 1_{[0, \infty)}(\frac{1}{2}) - 1_{[0, \infty)}(0) = 1 - 1 = 0.\]

  3. \[\lambda_{g_3}((-\infty, 0]) = 1_{[0, \infty)}(0) - \lim_{x \to -\infty} 1_{[0, \infty)}(x) = 1 - 0 = 1.\]

  4. \[\lambda_{g_3}((3,10]) = 1_{[0, \infty)}(10) - 1_{[0, \infty)}(3) = 0.\]

4. \(g_4(x) = x1_{[1,\infty)}(x)\).

  1. \[\lambda_{g_4}((0,2]) = 2*1_{[1,\infty)}(2) - 0*1_{[1,\infty)}(0) = 2 - 0 = 2.\]

  2. \[\lambda_{g_4}([0, \frac{1}{2}]) = \frac{1}{2}*1_{[1,\infty)}(\frac{1}{2}) - 0*1_{[1,\infty)}(0) = 0 - 0 = 0.\]

  3. \[\lambda_{g_4}((-\infty, 0]) = 0*1_{[1,\infty)}(0) - \lim_{x \to -\infty} x1_{[1,\infty)}(x) = 0 - 0 = 0.\]

  4. \[\lambda_{g_4}((3,10]) = 10*1_{[1,\infty)}(10) - 3*1_{[1,\infty)}(3) = 10-3 = 7.\]
